# 服务端渲染

## 基础的http服务
```javascript
// nodejs代码
const express = require('express')

// 获取express实例
const server = express()

// 编写路由处理不同url请求
server.get('/', (req, res)=>{
    res.send('<strong>hello world</strong>') // 返回
})

server.listen(80, () => {
    console.log('server running!')
})
```
